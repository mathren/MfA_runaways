#+Title: Live fast and die young: Massive, exploding, and speeding stars
#+author: [[mrenzo@flatironinstitute.org][Mathieu Renzo]]

This repo contains materials for a workshop with high-school teachers
run at [[https://www.mathforamerica.org/][Math for America]].

The folder [[./talk]] contains the source code for the slides of
[[file:talk/MfA_renzo_20230530.pdf][presentation]] (images and videos are not in the repo, but available on
request).

The folder [[./data]] contains materials for suggested exercises using
/Gaia/ third data release.
